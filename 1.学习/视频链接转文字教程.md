# 视频链接转文字教程

将视频链接转为可编辑文本的工作流，全程不超过10分钟。

## 一、两种方式

### 方式A：手动操作（✅ 经验证有效）

#### 第一步：复制视频链接
在 B站、抖音等平台，复制目标视频的浏览器地址栏网址（无需下载）。

#### 第二步：用 Snap Any 解析获取镜像地址

1. 访问 **https://snapany.com/bilibili**（注意是 `/bilibili` 不是 `/bili`）
2. 粘贴视频链接，点击「提取视频图片」
3. 弹出页面中**复制镜像网址**（不要点下载按钮）

#### 第三步：粘贴到通义听悟播客链接转写

1. 访问 **https://tingwu.aliyun.com** 并登录（每日送10小时免费转写）
2. 点击「**播客链接转写**」，粘贴镜像网址，点击「开始解析」
3. 等待转写（约5分钟），点击「导出」下载文本

---

### 方式B：API调用（❌ 暂时搁浅）

**问题**：[2026-05-14] 尝试调用通义听悟 CreateTask API（REST 风格，HMAC-SHA1签名），签名始终与服务器不匹配，调用失败。

**原因分析**：
- 通义听悟（tingwu）使用的是**项目制 AppKey/AppSecret**，而非阿里云标准 AccessKey
- AppKey 需要在听悟控制台创建项目后获取，不能用阿里云 AccessKey 直接调用
- 听悟 API 的签名算法为 REST 风格（Body-MD5 + 5段Header + 路径参数），手写实现始终对不上

**正确接入方式**：
1. 打开 https://tingwu.aliyun.com → 登录 → 左侧「**企业API**」→「创建项目」
2. 项目内有 **AppKey** 和 **AppSecret**（不是 AccessKey）
3. 用 AppKey/AppSecret 走阿里云 ROA 签名调用

**后续行动**：用户需提供 AppKey + AppSecret，AI方可正确调用。

## 三、AI调用建议（针对本工作流）

当用户要求"把这个视频转成文字"时，AI应：

1. **优先走手动流程**：告知用户复制视频链接，用 Snap Any + 听悟手动完成（3步）
2. **API方式**（等 AppKey 拿到后）：调用 `tingwu.cn-beijing.aliyuncs.com` 的 CreateTask 接口
3. **不要自行手写签名**：必须用阿里云 SDK（`pip install aliyunsdkcore`），用项目 AppKey 初始化 ROAClient

## 四、通用说明

- **抖音视频**：同样使用 https://snapany.com/bilibili，选择 TikTok 模式
- **适用场景**：个人学习、笔记整理，**禁止搬运商用**
- **效率**：手动方式全程不超过10分钟
