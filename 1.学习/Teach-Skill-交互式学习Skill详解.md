# Teach Skill — 交互式学习 Skill 详解

> 内容来源：抖音【见舟AI】，一个能让你快速掌握任何行业知识的 Codex Skill。

---

## 一、概述

**Teach** 是由海外开发者 **Matter** 打造的学习型 Skill，很多人称其为「人类满分程序员」。它擅长把复杂知识拆解成普通人也能理解的学习路径，核心价值不是「告诉答案」，而是像老师一样带人一步一步学会一个新领域。

---

## 二、使用方法

1. 打开 Codex
2. 把 Skill 地址丢给 AI 安装
3. 输入 `/teach` 即可使用

---

## 三、核心特点

### 3.1 结构化学习路径

以「从零学习 Agent 理论知识并做出自己的项目」为例，Teach 不会直接甩一堆资料，而是先搭建完整的学习工作区：

- **定义学习目标** — 从理解智能体到最终做出自己的 Agent 项目
- **整理可信参考资料**
- **搭建知识地图**
- **列出核心概念**
- **准备专业术语速查表**

### 3.2 互动式教学（最大亮点）

Teach **不是固定的教程，而是互动教程**：

- 学完一节课后会布置作业
- 例如它会问：「你最想做的 Agent 是什么？输入是什么？会调用哪些工具？什么时候应该停止？」
- 根据你的回答**重新生成下一节课**
- 每个人学到的内容都不一样——课程会随你的理解程度动态调整
- 你的问题、想法、项目方向都会被加入上下文，生成专属学习路线

### 3.3 学习闭环

```
学习一节 → 动手实践一步 → 反馈一次 → 进入下一轮
```

整个过程就像有一个 **AI 导师坐在旁边陪你学习**。

---

## 四、总结

Teach 最有价值的地方不是帮你查资料，而是帮你**建立学习闭环**——学一节、做一步、反馈一次、进入下一轮。每个人获得的学习路径都是个性化的。
